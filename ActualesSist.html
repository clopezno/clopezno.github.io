GII16.1 SWAPP MULTIPLATAFORMA Existen multitud de situaciones en las que se requiere un reparto de turnos o guardias entre un grupo de personas (e.g. personal sanitario en servicios hospitalarios, servicios de seguridad, emergencias, bomberos, etc.). En muchas ocasiones, el reparto de guardias planificado podría mejorarse fácilmente si se llevaran a cabo cambios bilaterales que a veces no se llegan a producir simplemente porque las partes implicadas desconocen que existe esa oportunidad de mejora conjunta. El objetivo de este proyecto es implementar una aplicación para teléfonos móviles que, partiendo de una distribución de guardias y un conjunto de preferencias declaradas por cada cliente: 1. Detecte cambios bilaterales que, en caso de producirse, incrementarían el bienestar de las personas implicadas (de acuerdo con su conjunto de preferencias declaradas). 2. De forma automática, mande un mensaje gratuito a las personas implicadas proponiendo el cambio y, 3. En el caso de que todas las partes implicadas acepten el cambio propuesto, actualice todos los calendarios que se han visto modificados por el cambio (así como una base de datos conjunta que idealmente pueda consultarse vía web). El cliente dispondrá de un calendario en el que deberá introducir sus guardias, sus preferencias (que como mínimo deben incluir un conjunto de guardias asignadas de las que el cliente se quiere desprender, así como un conjunto de turnos no asignados que el cliente estaría dispuesto a aceptar) y sus posibles restricciones (e.g. imposibilidad de llevar a cabo dos guardias seguidas). Naturalmente, todos estos datos deben poder ser modificables, y el sistema debe ser capaz de ajustarse automáticamente en tiempo real. Hace 2 años se propuso este proyecto para su implementación en Android, y se llegó a un prototipo operativo. Este proyecto es más ambicioso, en tanto que se pide el desarrollo de una aplicación multiplataforma. Para ello se recomienda usar Phonegap, un marco de desarrollo para aplicaciones móviles que permite codificar una sola vez usando JavaScript, HTML5, y CSS3, y compilar el código para que funciones en varias plataformas como iOS, Windows Phone, o Android. Luis R. Izquierdo Millán
